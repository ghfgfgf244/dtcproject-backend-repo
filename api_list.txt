DANH SÁCH API - DRIVING TRAINING CENTERS PROJECT (BACKEND)
==========================================================

1. AttendanceController (/api/Attendance)
-----------------------------------------
- [POST] /api/Attendance: Điểm danh cho học viên (Admin/TrainingManager/Instructor).
- [GET] /api/Attendance/Schedule/{classScheduleId}: Xem danh sách điểm danh theo buổi học.
- [GET] /api/Attendance/Report/Class/{classId}: Báo cáo điểm danh theo lớp.
- [GET] /api/Attendance/Student/{studentId}: Xem danh sách điểm danh của một học viên.
- [GET] /api/Attendance/Student/{studentId}/Summary: Xem tổng kết điểm danh của học viên (Số buổi học/vắng, Tỷ lệ %).

2. BlogController (/api/Blog)
-----------------------------
- [GET] /api/Blog: Danh sách tất cả bài viết (query ?onlyPublished=true để lọc đã xuất bản).
- [GET] /api/Blog/{id}: Xem chi tiết bài viết.
- [POST] /api/Blog: Tạo bài viết mới (Admin/TrainingManager).
- [PUT] /api/Blog/{id}: Cập nhật bài viết (Admin/TrainingManager).
- [DELETE] /api/Blog/{id}: Xóa mềm bài viết (Admin/TrainingManager).
- [PATCH] /api/Blog/{id}/publish: Xuất bản bài viết (Admin/TrainingManager).
- [PATCH] /api/Blog/{id}/unpublish: Ẩn bài viết (Admin/TrainingManager).
- [GET] /api/Blog/user/{userId}: Danh sách bài viết do một user tạo (sắp xếp mới nhất trước).

3. CategoryController (/api/Category)
--------------------------------------
- [GET] /api/Category: Danh sách tất cả danh mục.
- [GET] /api/Category/{id}: Xem chi tiết danh mục.
- [POST] /api/Category: Tạo danh mục mới (Admin/TrainingManager).
- [PUT] /api/Category/{id}: Cập nhật danh mục (Admin/TrainingManager).
- [DELETE] /api/Category/{id}: Xóa danh mục (Admin/TrainingManager).

4. AuthController (/api/Auth)
-----------------------------
- POST /api/Auth/sync: Đồng bộ thông tin người dùng từ Clerk vào Database.
- POST /api/Auth/logout: Đăng xuất.

3. CenterController (/api/Center)
---------------------------------
- POST /api/Center: Tạo trung tâm đào tạo mới - Admin.
- PUT /api/Center/{id}: Cập nhật thông tin trung tâm.
- DELETE /api/Center/{id}: Hủy kích hoạt trung tâm - Admin.
- GET /api/Center: Lấy danh sách tất cả các trung tâm.
- GET /api/Center/{id}: Xem chi tiết trung tâm.
- POST /api/Center/{id}/users: Gán người dùng vào trung tâm - Admin.

4. ClassController (/api/Class)
-------------------------------
- POST /api/Class: Tạo lớp học mới - Admin/TrainingManager.
- PUT /api/Class/{id}: Cập nhật thông tin lớp học.
- GET /api/Class: Lấy danh sách tất cả các lớp học.
- GET /api/Class/{id}: Xem chi tiết lớp học.
- DELETE /api/Class/{id}: Xóa lớp học.
- POST /api/Class/{id}/teachers: Gán giáo viên vào lớp.
- POST /api/Class/{id}/students: Gán học viên vào lớp.

5. CollaboratorController (/api/Collaborator)
---------------------------------------------
- GET /api/Collaborator/list: Xem danh sách cộng tác viên.
- GET /api/Collaborator/token: Lấy mã giới thiệu cá nhân.
- POST /api/Collaborator/token: Tạo mới mã giới thiệu.
- GET /api/Collaborator/token/usage: Xem số lượt sử dụng mã giới thiệu.
- GET /api/Collaborator/commission/rate: Xem tỷ lệ hoa hồng.
- POST /api/Collaborator/commission/calculate: Tính toán hoa hồng.
- GET /api/Collaborator/commission: Xem danh sách hoa hồng cá nhân.

6. CourseController (/api/Course)
---------------------------------
- POST /api/Course: Tạo khóa học mới - Admin/TrainingManager.
- PUT /api/Course/{id}: Cập nhật khóa học.
- DELETE /api/Course/{id}: Hủy kích hoạt khóa học.
- GET /api/Course/admin/all: Lấy tất cả khóa học (cho Admin).
- GET /api/Course/available: Lấy danh sách khóa học đang hoạt động.
- GET /api/Course/{id}: Xem chi tiết khóa học.

7. CourseRegistrationController (/api/CourseRegistration)
---------------------------------------------------------
- POST /api/CourseRegistration: Đăng ký khóa học (Học viên).
- PUT /api/CourseRegistration/{id}/cancel: Hủy đăng ký khóa học.
- PUT /api/CourseRegistration/{id}/status: Cập nhật trạng thái đăng ký (Duyệt/Từ chối).
- GET /api/CourseRegistration/me: Xem danh sách đăng ký của tôi.
- GET /api/CourseRegistration/all: Xem tất cả yêu cầu đăng ký.
- GET /api/CourseRegistration/{id}: Xem chi tiết yêu cầu đăng ký.

8. DashboardController (/api/Dashboard)
---------------------------------------
- GET /api/Dashboard/finance: Xem báo cáo tài chính.
- GET /api/Dashboard/admission: Xem báo cáo tuyển sinh/đào tạo.

9. DocumentController (/api/Document)
-------------------------------------
- POST /api/Document: Thêm thông tin giấy tờ (Metadata).
- POST /api/Document/upload: Upload trực tiếp file lên Cloudinary và lưu vào hệ thống.
- PUT /api/Document/{id}: Cập nhật thông tin giấy tờ.
- DELETE /api/Document/{id}: Xóa giấy tờ.
- GET /api/Document: Xem danh sách giấy tờ của tôi.
- GET /api/Document/{id}: Xem chi tiết giấy tờ (có link dynamic).
- POST /api/Document/{id}/verify: Xác thực giấy tờ - Admin.

10. EvaluationController (/api/Evaluation)
------------------------------------------
- POST /api/Evaluation: Tạo đánh giá học viên - Instructor.
- GET /api/Evaluation/{id}: Xem chi tiết đánh giá.
- GET /api/Evaluation/student/{studentId}: Xem tất cả đánh giá của một học viên.
- GET /api/Evaluation/class/{classId}: Xem tất cả đánh giá trong một lớp.
- PUT /api/StudentEvaluation/{id}: Cập nhật đánh giá của học viên.

- POST /api/StudentDrivingDistance: Tạo bản ghi quãng đường mới (Admin/TrainingManager).
- GET /api/StudentDrivingDistance/{id}: Lấy chi tiết bản ghi.
- PUT /api/StudentDrivingDistance/{id}/actual: Cập nhật quãng đường thực tế (Admin/TrainingManager/Instructor).
- GET /api/StudentDrivingDistance/my: Xem quãng đường của tôi (Học viên).
- GET /api/StudentDrivingDistance: Danh sách tất cả bản ghi (Admin/TrainingManager).
- DELETE /api/StudentDrivingDistance/{id}: Xóa bản ghi.

- DELETE /api/Evaluation/{id}: Xóa đánh giá.

11. ExamBatchController (/api/ExamBatch)
-----------------------------------------

- PUT /api/ExamBatch/{id}: Cập nhật kỳ thi.
- PATCH /api/ExamBatch/{id}/status: Cập nhật trạng thái kỳ thi (Đang mở/Đóng).
- DELETE /api/ExamBatch/{id}: Xóa kỳ thi.

12. ExamController (/api/Exam)
------------------------------
- POST /api/Exam: Tạo bài kiểm tra.
- GET /api/Exam/{id}: Xem chi tiết bài kiểm tra.
- PUT /api/Exam/{id}: Cập nhật bài kiểm tra.
- DELETE /api/Exam/{id}: Xóa bài kiểm tra.
- GET /api/Exam: Xem danh sách bài kiểm tra.
- GET /api/Exam/{id}/results: Xem kết quả bài kiểm tra.
- PUT /api/Exam/{id}/results/{resultId}: Cập nhật kết quả bài thi.
- POST /api/Exam/results/bulk: Nhập kết quả thi hàng loạt.

13. ExamRegistrationController (/api/ExamRegistration)
------------------------------------------------------
- POST /api/ExamRegistration: Đăng ký thi (Học viên).
- PATCH /api/ExamRegistration/{id}/status: Cập nhật trạng thái đăng ký thi.
- PATCH /api/ExamRegistration/{id}/pay: Xác nhận đã thanh toán phí thi.
- GET /api/ExamRegistration/Batch/{examBatchId}: Lấy danh sách đăng ký theo kỳ thi.
- GET /api/ExamRegistration/Student/{studentId}: Lấy danh sách đăng ký theo học viên.
- POST /api/ExamRegistration/bulk: Tạo đăng ký thi hàng loạt.

14. FileController (/api/File)

22. UserController (/api/users)
-------------------------------
- GET /api/users/me: Xem hồ sơ cá nhân.
- PUT /api/users/me: Cập nhật thông tin cá nhân.
- POST /api/users/me/apply-staff: Ứng tuyển làm giáo viên hoặc cộng tác viên.

- POST /api/users: Tạo người dùng mới - Admin.

- GET /api/users/instructors: Lấy danh sách toàn bộ giáo viên.

- PUT /api/users/{id}/roles: Cập nhật quyền hạn cho người dùng - Admin.
- GET /api/users: Lấy danh sách tất cả người dùng - Admin.
- PUT /api/users/{id}/toggle-status: Khóa hoặc mở khóa tài khoản.
